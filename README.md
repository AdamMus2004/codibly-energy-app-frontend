# Energy Mix UK - Frontend Dashboard

---

## Project Overview

Frontend dashboard for the Energy Mix UK application.
Visualizes energy generation data using charts and provides an interface for the EV charging optimizer.

---

## Tech Stack
* **React 18**
* **TypeScript**
* **Vite**
* **Recharts**
* **Axios**

## Running the Project

### Local (npm)
```bash
npm install
npm run dev
```
The application will start at http://localhost:5173.

---

## Configuration
The app connects to the backend API:
* Local: Defaults to http://localhost:8080
* Production: Uses VITE_API_URL environment variable.
